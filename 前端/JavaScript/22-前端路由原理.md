# 前端路由

## hash路由
![images](images/router/router01.png)
```html
<body>
  <div>
    <a href="#home">首页</a>
    <a href="#about">关于</a>
    <a href="#detail">详情</a>
  </div>
  <div id="container"></div>
  <script>
    window.addEventListener('hashchange', (e) => {
      let container = document.getElementById('container');
      container.innerText = location.hash;
    })
  </script>
</body>
```
